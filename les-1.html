Les 1 - ISM Workshop - Basic TCP - HTTP

In deze workshop ga je aan de slag met enkele internet protocollen. Oftewel: how works them Interwebs?

Er zijn 3 mogelijkheden, kies degene die je het handigst lijkt.

1. Packet Tracer installeren

  • Download PacketTracer.zip
  • Download PacketTracer.z01
  • Download PacketTracer.z02
  • Download WebtechAssignment.pkt
  • Pak de .zip uit
  • Dubbelklik op de uitgepakte .exe om Packet Tracer te installeren
  • Open de 
    WebTechAssignment.pkt
    en volg de verdere stappen uit de Powerpoint met 'prerequisites' voor Les 1

2. Virtual Box gebruiken en de .vdi bestanden

Dit is sowieso nodig voor 2 Workshops einde course, maar vereist wel ca. 30 GB vrije HD ruimte.

  • Download virtualbox.org
  • Download virtualbox.org
  • Open de 
    WebTechAssignment.pkt
    en volg de verdere stappen uit de Powerpoint met 'prerequisites' voor Les 1

3. Docker

Onderstaande ou wellicht ook moeten werken voor teh Docker loverz (ongetest)

Zorg zelf dat je de .pkt download en op de Docker instance kopieert en kunt openen.

Bron/verdere info: https://github.com/somatorio/docker-packet-tracer

    docker run -it --rm \
      -e DISPLAY=unix$DISPLAY \
      -v /tmp/.X11-unix/:/tmp/.X11-unix/ \
      -v $HOME/.packettracer:/root/pt \
      somatorio/packet-tracer
